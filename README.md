# PID_Copilot_GINTE2L_T1

Repositorio del **Taller 1** del PID (Proyecto de Innovación Docente) del GID **GINTE2L** — *Aplicaciones de IA generativa en la docencia universitaria*.

## Objetivos
- Introducir buenas prácticas de uso de **Microsoft Copilot** y otras IA generativas en docencia.
- Mostrar ejemplos aplicados a **Electrónica** (generación de ejercicios, prácticas guiadas, feedback automatizado).
- Proveer materiales, rúbricas y plantillas reutilizables.

## Estructura prevista
- `docs/` — documentación del taller (programa, guía de instalación, referencias).
- `materiales/` — presentaciones y materiales entregables.
- `ejemplos/` — notebooks y scripts de demostración.
- `recursos/` — plantillas de evaluación y rúbricas.
